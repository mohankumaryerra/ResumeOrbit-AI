# ResumeAI – AI-Powered Resume Analyzer  ResumeAI is a modern web application that helps job seekers optimize their resumes using AI-powered analysis. It evaluates resumes for ATS (Applicant Tracking System) compatibility, identifies keyword gaps, improves bullet points, and provides personalized career recommendations
## ✨ Features

- 📄 Upload Resume (PDF/DOCX/TXT)
- 🤖 AI-Powered Resume Analysis using Gemini AI
- 📊 ATS Score Evaluation
- 🔍 Keyword Gap Detection
- ✍️ Bullet Point Enhancement
- 📝 Grammar & Clarity Suggestions
- 💼 Recruiter-Friendly Resume Feedback
- 🎯 Personalized Career Recommendations
- ❓ Interview Preparation Suggestions
- 🌙 Modern Responsive UI
- ⚡ Fast Analysis Experience

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Gemini AI API
- HTML5
- CSS3

## 🚀 Purpose

The goal of ResumeAI is to help students, freshers, and professionals build ATS-friendly resumes and improve their chances of getting shortlisted by recruiters.
